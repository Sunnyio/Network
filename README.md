# Network: A Social Media Application

Network is a social media application that allows users to create profiles, follow other users, and share posts, similar to Twitter. Built with Python and Django on the backend, and JavaScript, HTML, and CSS on the frontend, it offers a responsive and engaging user experience.

## Features

- **User Authentication**: Secure user registration and login functionality.
- **Profile Management**: Users can create and edit their profiles.
- **Following System**: Follow and unfollow other users to customize your feed.
- **Post Creation**: Compose, edit, and delete text-based posts.
- **Like Functionality**: Like and unlike posts to engage with content.
- **Pagination**: Browse through posts with seamless pagination.
